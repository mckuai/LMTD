Diferencia Media Logaritmica de Temperatura (LMTD)¶

La diferencia media logartmica de temperatura est dada por la siguiente frmula:

$$ LMTD = \frac{(T_{1}-t_{2}) - (T_{2}-t_{1})}{ln \left(\frac{T_{1}-t_{2}}{T_{2}-t_{1}}\right)} $$$$ LMTD = \frac{(\Delta T_{1}) - (\Delta T_{2})}{ln \left(\frac{\Delta T_{1}}{\Delta T_{2}}\right)} $$

Dejando de lado la teora que acompaa a esta frmula, la tarea es conseguir que con una $LMTD$ y una Temperatura de entrada $(T_{1})$ conocidas, obtener el resto de variables tales que LMTD se cumpla.

Esto es con el objetivo de que se observe el cambio en el resto de variables conforme $T_{1}$ cambia y $LMTD$ permanece fija

Para resolver el problema en cuestin, con ayuda de Python 3, se me ocurren dos opciones para afrontar el problema:

  1. Mtodos numricos
  2. Probar con todas las permutaciones en un intervalo $[1-T_{1}]$

La ventaja es que los valores conocidos para $T_{1}$ no rebasan los 30. Sin duda puedo reducir los clculos introduciendo algunas restricciones. Solución manual¶

He trabajado en una solucin manual que satisface en cierta medida los requerimientos del problema. El objetivo ultimo, es que esta tarea se automatice.

En el cdigo de abajo se introducen las variables y se obtiene la $LMTD$ ademas de los deltas y una representacin grfica.

import numpy as np
import matplotlib.pyplot as plt
import pandas as pd
from ipywidgets import interactive

%matplotlib inline

def Tml_contra(T_hs_in,T_hs_out, T_f_in, T_f_out):
    delta_T1 = T_hs_in - T_f_out
    delta_T2 = T_hs_out - T_f_in
    
    Tmlc = (delta_T1-delta_T2)/np.log(delta_T1/delta_T2)
    
    #Generacin del grfico
    fig1, ax1 = plt.subplots()
    ax1.plot([T_hs_out,T_hs_in], 'r--')
    ax1.plot([T_f_in, T_f_out], 'b--')
    ax1.set_ylabel(' T C ', fontsize = 12)
    
    ax2 = ax1.twinx()
    ax2.plot([T_hs_out,T_hs_in], 'r<', markersize = 8, label = 'Calor')
    ax2.plot([T_f_in, T_f_out], 'b>', markersize = 8, label = 'Frijolito')
    ax2.set_ylabel(' T C ', fontsize = 12)
  
    ax2.annotate('Tmlc: '+str(round(Tmlc,2)),
            xy=(0.3 , T_hs_in), xycoords='data',
            xytext=(0.3, T_hs_in), textcoords='data')
    ax2.legend()
    
    plt.title('Tml contra corriente')
    plt.grid()
    plt.xticks([])
    plt.show()
    
    msg = f"La TML es: {delta_T1} Delta T1 es:{delta_T1} Delta T2 es:{delta_T2}"
    msg1 = f"La TML es: {Tmlc} Delta T1 es:{delta_T1} Delta T2 es:{delta_T2}"

    if delta_T1 == delta_T2:
        return msg
    else:
        return msg1

Donde:

  • Tml_contra = Tmlc = $LMTD$
  • T_hsin = $T{1}$
  • T_hsout = $T{2}$
  • T_fin = $t{1}$
  • T_fout = $t{2}$

El objetivo es que mediante la introduccin de valores diferentes de entrada (de manera manual) se obtuviera la $LMTD$ deseada.

Como ejemplo, para una $LMTD$ deseada de $\approx 5$ se realiza lo siguiente:

$$ \frac{(T_{1}-t_{2}) - (T_{2}-t_{1})}{ln \left(\frac{T_{1}-t_{2}}{T_{2}-t_{1}}\right)} \approx 5$$ 
T_hs_in = 26
T_hs_out = 15

T_f_in = 8
T_f_out = 22

Comprobación¶

Introduciendo los valores generados en una funcin que calcule la $LMTD$ podemos comprobar si la solucin es adecuada.

Función para calcular LMTD¶

def LMTD(T1,T2,t1, t2):
    delta_T1 = T1 - t2
    delta_T2 = T2 - t1
    
    try:
        Tmlc = (delta_T1-delta_T2)/np.log(delta_T1/delta_T2)
    except:
        return NaN
    
    if delta_T1.any() == delta_T2.any():
        return delta_T1
    else:
        return Tmlc
